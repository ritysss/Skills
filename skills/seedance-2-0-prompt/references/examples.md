# Seedance 2.0 提示词示例

这些示例用于参考结构、密度和字段写法。除非用户要求相似场景，不直接复制具体内容。

## 文生视频

```text
【基础设定】
雨夜街口，一名穿黑色风衣的女性停在霓虹招牌下，缓慢回头看向镜头。

【分镜与景别】
中近景，人物位于画面中央偏右，前景有虚化雨滴，背景是湿润街面和模糊车灯。

【提示词】
[主体] 25岁女性，黑色短发，黑色风衣，银色耳饰，站在雨后的城市街口，地面有霓虹反射。
[动作] 她先停住脚步，右手轻轻压住风衣领口，随后缓慢回头并与镜头对视。
[镜头] 85mm 中长焦半身特写，浅景深，摄影机轻微横移，速度缓慢，云台稳定，焦点锁定人物眼睛。
[风格] 写实电影感，左侧品红色霓虹边缘光，右侧冷蓝环境光，低调光，高对比阴影，35mm胶片质感，自然颗粒，湿润街面反光清晰，柔和高光溢出。
[约束] 人物外观一致，面部结构稳定，自然人体比例，动作连贯克制，画面平稳，色彩基调一致，主体不漂移。
[音频] 无BGM，保留雨声、远处车流声和轻微衣料摩擦声。
```

## 多参考素材

```text
【基础设定】
小型livehouse舞台上，女歌手在副歌开始时抬手靠近麦克风，准备唱出第一句。

【参考素材使用】
@图片1 作为人物外观、服装和首帧构图参考；@视频1 仅参考缓慢推进的运镜节奏；@音频1 作为副歌节拍和人声口型参考。

【分镜与景别】
中景转中近景，人物位于画面中心，前景保留虚化麦克风，背景是观众轮廓和舞台灯架。

【提示词】
[主体] 女歌手穿银色亮片短外套，手持黑色麦克风，站在蓝紫色舞台灯下，背景观众保持次要。
[动作] 她跟随副歌第一拍轻抬右手，身体向前半步，唱出一句歌词后停顿并看向镜头。
[镜头] 50mm 标准镜头中景缓慢推进到中近景，云台稳定，主体锁定，焦点从麦克风自然过渡到人物眼睛。
[风格] 演唱会写实电影感，蓝紫色舞台主光，暖色轮廓光勾勒肩部，轻微镜头炫光，肤色自然，轻微胶片颗粒，舞台烟雾与灯光层次清晰。
[约束] 人物外观与@图片1一致，动作不抢拍，面部结构稳定，舞台背景不抢主体，画面平稳，色彩不跳变。
[音频] @音频1 作为节奏与人声参考，单一演唱者口型尽量贴合主旋律，无额外BGM。
```

## 首尾帧

```text
【基础设定】
一辆红色跑车从雨夜街口缓慢驶出，最终停在亮着白色灯牌的店门前。

【参考素材使用】
@图片1 作为首帧车身外观和起始构图参考；@图片2 作为尾帧停车位置、灯牌方向和最终构图参考。

【分镜与景别】
低机位中景，车身占画面下方三分之二，街道反光形成前景引导线。

【提示词】
[主体] 红色跑车，车身漆面湿润反光，雨后城市街道，远处有冷蓝路灯和暖色店铺灯。
[动作] 跑车从静止状态缓慢启动，前轮轻微转向，沿湿润路面向前滑行，最后平稳停在店门前。
[镜头] 35mm 广角低机位跟拍，摄影机从前侧45度平稳横移，速度缓慢，云台稳定，焦点锁定车头与车标。
[风格] 高端汽车商业广告质感，冷蓝街灯与暖色店铺灯形成冷暖对比，湿地反射清晰，高光不过曝，产品级锐度，金属与漆面反射清晰，低噪点。
[约束] 车身颜色和线条与@图片1一致，尾帧构图贴近@图片2，运动平稳，反射稳定，背景不重构，主体不漂移。
[音频] 无BGM，保留低沉引擎声、轮胎压过湿地的细微水声和远处城市环境声。
```

## 视频编辑

```text
【基础设定】
原视频中的街头行走镜头被局部改成新角色经过橱窗，整体空间和原始运镜保持一致。

【参考素材使用】
@视频1 作为原始视频，保留时长、运镜节奏、整体光线和街道结构；@图片1 作为替换角色的外观和服装参考。

【分镜与景别】
沿用@视频1的中景街头跟拍构图，主体位于画面中心偏左，橱窗反光位于画面右侧。

【提示词】
[主体] 将原主角替换为@图片1中的角色，保留街道、橱窗、路灯、背景行人和整体空间关系。
[动作] 替换角色按照原视频节奏从画面左侧向前行走，经过橱窗时轻微转头看向反光。
[镜头] 完全沿用@视频1的原始运镜，不新增推拉、摇镜或快速变焦，焦点保持在替换角色面部与上半身。
[风格] 延续@视频1的写实电影质感，原始色温、曝光、阴影关系、颗粒、锐度和运动模糊保持一致，橱窗反光与街灯亮度自然。
[约束] 未指定区域保持稳定，替换角色外观与@图片1一致，街道结构不改变，剪辑节奏不改变，画面平稳。
[音频] 无BGM，保留原视频环境声；如原视频无声，则静音处理。
```

## 多镜头分镜

```text
【基础设定】
清晨厨房里，一杯热咖啡被端到窗边，窗外雨水模糊了城市街景。

【分镜】
镜头1 [0:00-0:04]：微距 - 咖啡表面的热气缓慢上升；微距镜头固定机位，焦点锁定咖啡泡沫；暖色室内光。
镜头2 [0:04-0:08]：中近景 - 手端起咖啡杯走向窗边；35mm 广角轻微跟拍，云台稳定；室内暖光过渡到窗外冷蓝。
镜头3 [0:08-0:12]：特写 - 咖啡杯停在窗台，焦点从杯沿转移到窗外雨滴；85mm 中长焦浅景深，三脚架稳定。
转场：镜头1到镜头2硬切；镜头2到镜头3动作匹配剪辑。

【提示词】
[主体] 白色陶瓷咖啡杯，木质窗台，清晨室内厨房，窗外是雨天城市街景。
[动作] 热气上升，手端起杯子并放到窗台，动作轻缓自然。
[镜头] 每个镜头只保留一个核心运镜；焦点变化缓慢，景深过渡自然，画面保持稳定。
[风格] 写实生活电影感，室内暖黄色柔光与窗外冷蓝雨景形成冷暖对比，曝光连续，白平衡稳定，柔和胶片颗粒，陶瓷、木纹和雨滴质感清晰。
[约束] 杯子外观一致，空间关系连续，色彩基调统一，转场自然，动作节奏平稳。
[音频] 无BGM，保留咖啡杯轻放声、室内底噪和窗外细雨声。
```
