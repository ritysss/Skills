# NanoBanana Pro 摄影术语词库

适用于静态图片提示词。所有术语均为英文，直接用于提示词中。

---

| 术语 | 描述 | 适用场景 |
|------|------|---------|
| `14mm ultra-wide` | 超广角，强透视畸变 | 建筑内部、夸张空间感 |
| `24mm wide-angle` | 广角 | 风景、环境人像、街拍 |
| `35mm standard wide` | 标准广角 | 街头摄影、纪实、叙事场景 |

| `105mm medium telephoto` | 中长焦 | 人像特写、细节 |

## 本地样本标注（2026-03-12 Higgsfield）

以下标注基于 5 个测试、13 张出图，只反映当时 Higgsfield 路径中的小样本观察。把它当作选词启发，不要当成 Google 原生模型的命中率或机制保证。

### 实测验证有效的术语组合

| 术语 | 测试场景 | 效果 |
|------|---------|------|
| `Kodak Portra 400` | 电影感人像 | ★★★★★ 色彩科学精确，暖肤色+自然颗粒一致 |
| `Fuji Pro 400H` | 多主体拉面店 | ★★★★★ 柔和提亮暗部、淡雅色调精准 |
| `golden hour light` + 方向描述 | 人像/美食 | ★★★★★ 光线角度和色温完全正确 |
| `neon magenta and cyan` | 暗调海报 | ★★★★★ 霓虹色彩精确还原 |
| `tungsten + fluorescent mix` | 多主体室内 | ★★★★★ 暖黄+冷蓝双色温分区清晰 |
| `85mm portrait lens, f/1.8` | 人像 | ★★★★☆ 背景压缩和虚化正确 |
| `28mm wide-angle` | 多主体室内 | ★★★★☆ 广角畸变感到位 |
| `100mm macro lens` | 美食微距 | ★★★★☆ 微距细节优秀 |
| `translucent amber` | 蜂蜜材质 | ★★★★★ 半透明光线穿透感极佳 |
| `weathered / tarnished` | 做旧金属 | ★★★☆☆ 做旧程度偏弱，倾向渲染为较新状态 |
| `isometric 3D render` | 悬浮岛 | ★★★★★ 风格切换干净，零写实残留 |
| `stylized feature-animation quality` | 3D 场景 | ★★★★★ 该样本中呈现清晰的玩具化动画质感 |

### 实测效果弱的术语/描述方式

| 术语/描述 | 问题 | 替代建议 |
|----------|------|---------|
| `top-down overhead` | 模型自动修正为 45° 斜俯 | 用 `strictly 90-degree bird's-eye, camera pointing straight down` |
| `three-quarter angle` | 容易偏向正面或侧面 | 追加具体度数 `camera positioned at 45-degree horizontal angle` |
| `lips barely parted` | 基本不执行 | 省略或改为更明显的动作 |
| `gazes slightly past the lens` | 视线控制不精确 | 改为明确方向 `looking to the right` |
| `f/2.8 focus on [specific subject]` | 多主体时无法选择性聚焦 | 接受全部清晰，或减少主体数量 |

## 语义负向替代表（正向锚定）

NanoBanana Pro **不支持 negative_prompt**。要排除元素，用正向描述替代：

| ❌ 避免（负面/排除） | ✅ 使用（正向替代） |
|---------------------|-------------------|
| no cars / remove vehicles | an empty, deserted street |
| don't blur the face | sharp focus on the subject's face |
| no noise / no grain | clean, noise-free image |
| avoid overexposure | well-balanced, even exposure |
| don't crop the body | full-body framing with generous headroom |
| no distracting background | clean, minimal background |
| not too saturated | muted, natural color palette |
| don't make it dark | bright, well-lit scene |
| no wrinkles on the fabric | smooth, freshly pressed fabric |
| avoid flat lighting | dynamic lighting with clear shadows |
| don't make it look artificial | photorealistic, natural appearance |
| no lens distortion | optically corrected, distortion-free |
| not too wide | standard 50mm focal length |
| no text in the image | clean image without any typography |
| no people | a solitary, uninhabited landscape |
| avoid clutter | organized, intentional arrangement |
