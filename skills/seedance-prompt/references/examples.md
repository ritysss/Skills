# Seedance Prompt Examples

These examples show shape and density. Do not copy content unless the user asks for a similar scene. Single-shot examples use only the six required fields; reference roles are declared inside their owning fields.

## Text-To-Video

```text
[主体] 夜晚雨后的城市街口，一名25岁黑色短发女性穿黑色风衣站在霓虹招牌下，右手靠近领口，双脚踩在同一湿地反光位置。
[动作] 她先停住脚步，右手轻压风衣领口，肩膀微微绷紧，随后缓慢回头并与镜头对视，最后保持克制警觉的呼吸。
[镜头] 中近景，人物位于画面中央偏右，前景保留虚化雨滴，背景是湿润街面和模糊车灯；85mm中长焦，眼平机位，摄影机因人物回头而轻微横移半个身位，云台稳定，焦点锁定眼睛。
[风格] 写实电影感，左侧品红霓虹形成边缘光，右侧冷蓝环境光压住阴影，肤色自然，湿地反射清晰，细腻胶片颗粒和柔和高光溢出。
[约束] 面部结构、体型比例和服装稳定，双脚保持同一地面接触点，动作连贯克制，白平衡和曝光连续。
[音频] 无BGM，保留雨声、远处车流声和衣料摩擦声。
```

## Multimodal Reference

```text
[主体] 小型livehouse舞台上，@图片1锁定女歌手的面孔、银色亮片短外套和黑色麦克风；她站在舞台中央，背景观众保持次要。
[动作] @视频1只控制缓慢抬手、向前半步和停顿的表演节奏；女歌手跟随副歌第一拍抬起右手，靠近麦克风唱出一句歌词，随后停住并看向镜头。
[镜头] 中景起始构图，人物位于画面中心，前景保留虚化麦克风，背景是观众轮廓和灯架；@视频1只控制缓慢推进的运镜节奏，50mm标准镜头、眼平机位，云台稳定，焦点从麦克风转到眼睛。
[风格] 蓝紫色舞台主光与暖色肩部轮廓光形成分离，肤色自然，轻微镜头炫光和颗粒，舞台烟雾只在光束内可见。
[约束] 人物身份和服装以@图片1为权威，@视频1只承担表演与运镜节奏，面部结构稳定，动作贴拍，舞台背景不抢主体。
[音频] @音频1锁定副歌节拍和单一演唱者的人声口型，画面动作贴合主拍点，无额外BGM。
```

## First/Last-Frame

```text
[主体] 雨夜城市街道上，@图片1锁定红色跑车的车身身份、漆面材质和起始状态，远处冷蓝路灯与暖色店铺灯同时可见。
[动作] 跑车从静止状态缓慢启动，前轮轻微转向，沿湿润路面平稳滑行，最终在店门前一次停稳。
[镜头] @图片1锁定首帧车身位置和原始透视，@图片2锁定尾帧停车位置、灯牌方向和最终构图；中景低机位，车身占画面下方三分之二，摄影机从前侧45度平稳横移跟拍，焦点锁定车头与车标。
[风格] 高端汽车商业广告质感，冷蓝街灯与暖色店铺灯形成冷暖对比，湿地反射清晰，金属和漆面高光受控，产品级锐度和低噪点。
[约束] 车身颜色、比例和线条以@图片1为权威，运动路径唯一，反射与街道结构稳定，@图片2锁定尾帧可见状态。
[音频] 无BGM，保留低沉引擎声、轮胎压过湿地的细微水声和远处城市环境声。
[终帧] @图片2锁定跑车平稳停在店门前的最终构图、车头与白色灯牌方向；焦点锁定车标，车身完整清晰。
```

## Edit Video

```text
[主体] @视频1锁定原街头视频的时长、街道、橱窗、路灯、背景行人和整体空间；@图片1只锁定替换角色的面孔、发型和服装。
[动作] 替换角色沿用@视频1原人物的行走节奏从画面左侧向前移动，经过橱窗时只轻微转头看向反光。
[镜头] 中景街头跟拍，主体保持画面中心偏左，橱窗反光位于右侧；完全继承@视频1的原始构图、透视、机位、运镜阶段和焦点关系。
[风格] 继承@视频1的原始色温、曝光、阴影、颗粒、锐度和运动模糊，替换角色自然融入街灯与橱窗反射。
[约束] 只替换角色，街道结构、背景人物、镜头节奏和未指定区域保持原始可见状态；角色身份以@图片1为权威。
[音频] 沿用@视频1原始环境声；源视频无声时保持静音，无新增BGM。
```

## Extension From Accepted Take

```text
[主体] @视频1锁定最新已接受成片的结尾状态：火车站月台上的旅客已经扣好行李箱锁扣，左脚踏实、右膝弯曲、右手刚离开锁扣；@图片1只锁定旅客面孔、发型和服装。
[动作] 她从现有蹲姿开始，左手压住左膝，重心移到左脚并一次连续站起；站稳后右手握住箱杆并拉到腰部高度，最后转头看向画面左侧的列车进站方向。
[镜头] @视频1锁定中景起始构图、人物右三分之一位置、50mm镜头关系、眼平机位和缓慢左移的当前阶段；摄影机方向与速度连续，焦点从锁扣平稳转到眼睛。
[风格] 继承@视频1的清晨自然光、曝光、色温、月台空气感和金属反射。
[约束] 锁扣保持已经闭合的状态，本段只执行起身、拉杆和转头这一条新动作路径；人物、行李箱和月台空间连续。
[音频] 沿用@视频1环境声底，保留远处列车声、衣料摩擦、鞋底受力声和箱杆机械声，无新增BGM。
[终帧] 旅客直立停在画面右三分之一，右手握住腰部高度的箱杆，头部和视线指向画面左侧，摄影机左移结束，焦点锁定眼睛。
```

## Sequential Cuts Without Timecodes

```text
【分镜】
镜头1：中景，@图片1锁定白色陶瓷杯、木质料理台和厨房空间；一只手端起热咖啡，固定机位，焦点锁定杯沿热气。
镜头2：中近景，同一只手把咖啡端向窗边；35mm广角平稳跟拍，动作方向保持由右向左，室内暖光逐渐让位于窗外冷蓝雨光。
镜头3：特写，咖啡杯一次轻放在窗台；85mm中长焦固定机位，焦点从杯沿转到窗外雨滴。
转场：镜头1到镜头2使用动作匹配剪辑，镜头2到镜头3使用硬切；只在上述位置切镜。

【跨镜头一致性】
@图片1锁定白色陶瓷杯、木质窗台和厨房空间跨镜头一致；同一只手、杯中液位、动作方向、曝光和白平衡连续。

【音频】
无BGM，保留咖啡轻微沸腾声、杯子轻放声、室内底噪和窗外细雨声。
```

## Timed Multi-Shot

```text
【分镜】
镜头1 [0:00-0:04]：微距，咖啡表面的热气缓慢上升；固定机位，焦点锁定泡沫；暖色室内光，保留轻微沸腾声。
镜头2 [0:04-0:08]：中近景，手端起咖啡杯走向窗边；35mm广角轻微跟拍，云台稳定，室内暖光过渡到窗外冷蓝。
镜头3 [0:08-0:12]：特写，咖啡杯轻放在窗台；85mm中长焦浅景深，焦点从杯沿转到窗外雨滴，三脚架稳定。
转场：镜头1到镜头2硬切，镜头2到镜头3动作匹配剪辑；只在上述时间点切镜。

【跨镜头一致性】
白色陶瓷杯、木质窗台和厨房空间保持一致；曝光、白平衡、杯中液位和动作方向连续。

【音频】
无BGM，保留咖啡轻微沸腾声、杯子轻放声、室内底噪和窗外细雨声。
```
